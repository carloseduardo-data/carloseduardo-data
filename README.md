## Projetos em Destaque

### BI de Avanço Semanal de Obra
Dashboard desenvolvido no Power BI para acompanhamento de Programado x Realizado com base nos dados do RDO extraídos do ERP. Modelagem e consultas em SQL para estruturação do banco de dados e consolidação das informações, com foco em clareza visual (UI) e suporte à tomada de decisão.

### Automação de Processos Administrativos
Desenvolvimento de automações em Python para otimização de rotinas administrativas e consolidação de dados. Utilização de bibliotecas como Pandas e NumPy para transformação de dados, Requests para integração com APIs, OpenPyXL e Xlwings para manipulação de arquivos Excel e SQLAlchemy / PyODBC para conexão com bancos de dados, garantindo padronização, confiabilidade e redução de atividades manuais.

### Pipeline ETL
Construção de pipelines de extração, transformação e carga de dados utilizando Python e SQL para integração de múltiplas fontes, tratamento de inconsistências e estruturação de bases consolidadas para análise e tomada de decisão.
